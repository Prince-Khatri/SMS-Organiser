# SMS Organizer

SMS Organizer is an Android application that helps organize SMS messages into useful categories like Personal, Transactions, Promotions, Spam, etc.

## Development Status

The project is currently in the development phase.

### Completed
- Basic frontend UI
- RecyclerView chat-style SMS layout
- SMS fetching from device
- Room database integration
- Message card showing sender, message body, time, and category chip
- Frontend navigation setup

### In Progress
- Message categorization logic

### Planned
- Machine learning based SMS classification
- Organisation of messages based on preferences
- Search and filtering

## Tech Stack
- Java
- Android SDK
- RecyclerView
- Room Database
